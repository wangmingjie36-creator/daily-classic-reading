# 📚 每日经典精读 · 自进化学习系统

> 一个由 AI 驱动的**跨学科阅读→思维模型提炼→自我进化**系统。
> 每天自动精读一本经典，提炼思维模型卡片，每周发现跨书共鸣，每月自动创建新技能。

---

## 这是什么？

这是一个运行在 [Claude Cowork](https://claude.ai) 上的自动化学习系统，包含三个层次：

| 层次 | 频率 | 做什么 |
|------|------|--------|
| 🔵 每日精读 | 每天 8:00 | 自动选书、生成深度精读 Markdown、提炼思维模型卡片 |
| 🟡 每周元学习 | 周日 21:00 | 发现跨书共鸣与张力、更新知识图谱、生成周报 |
| 🔴 每月自进化 | 每月 1 日 | 回顾所有周报、自动创建新 Skill、优化 Prompt |

系统的核心理念：**学习不是堆积信息，而是发现不同知识之间的深层连接。**

## 快速开始

### 前置条件

- [Claude Desktop](https://claude.ai/download) + Cowork 模式
- 创建一个文件夹（比如 `~/每日阅读经典`），在 Cowork 中选择它作为工作目录

### 第 1 步：初始化思维模型库

把本仓库的 `思维模型库/` 文件夹复制到你的工作目录中。如果你想从零开始，只保留以下空结构：

```
思维模型库/
├── INDEX.md          ← 可以清空内容，保留模板
├── cards/            ← 空目录
├── skills/           ← 保留两个 Skill
├── memory/
│   ├── learning_memory.json  ← 重置为初始状态（见下方）
│   └── knowledge_graph.json  ← 重置为初始状态
├── 周报/
└── 月报/
```

初始 `learning_memory.json`：
```json
{
  "version": "1.1",
  "created": "你的开始日期",
  "last_updated": "你的开始日期",
  "daily_signals": [],
  "category_balance": {"哲学思想": 0, "科学认知": 0, "历史社科": 0, "文学名著": 0},
  "card_distribution": {"P_底层原理": 0, "H_人性行为": 0, "M_思维模型": 0, "S_社会结构": 0},
  "user_interest_signals": {},
  "task_health": {},
  "skill_candidates": [],
  "evolution_history": []
}
```

### 第 2 步：创建定时任务

在 Cowork 中对 Claude 说：

> 帮我创建一个每天早上 8 点的定时任务，名叫 daily-classic-reading

然后把 `prompts/daily-classic-reading.md` 的内容粘贴给它。

同样创建：
- `weekly-meta-learning`（周日 21:00）
- `monthly-self-evolution`（每月 1 日 10:00）

### 第 3 步：等待系统运转

- **第 1 天**：收到第一篇精读 + 第一批思维模型卡片
- **第 7 天**：收到第一份周报，看到跨书共鸣开始浮现
- **第 30 天**：月度自进化报告，系统自动创建新 Skill

## 思维模型卡片体系

每本书的洞见被提炼为四种卡片：

| 类型 | 代号 | 含义 | 示例 |
|------|------|------|------|
| 底层原理 | P | 跨学科的普适规律 | P-001 节能默认律 |
| 人性行为 | H | 认知偏误、动机、自欺 | H-001 理性驾驶员幻觉 |
| 思维模型 | M | 可直接调用的心智工具 | M-002 按下暂停键 |
| 社会结构 | S | 制度、权力、文明演化 | S-001 示弱作为进攻策略 |
| 元卡片 | META | ≥3本书的深层同构 | META-001 个体是载具假说 |

卡片之间通过「深化/配合/张力/来源」四种关系形成知识网络。

## 自定义

### 调整阅读方向

编辑 `prompts/daily-classic-reading.md` 中的「安全书单」，替换为你感兴趣的书目。

### 调整卡片维度

如果你的兴趣偏向其他方向（比如技术/艺术/心理学），可以修改四大维度的定义。

### 添加新 Skill

当你发现反复出现的思维模式，可以在 `思维模型库/skills/` 下创建新的 SKILL.md。
