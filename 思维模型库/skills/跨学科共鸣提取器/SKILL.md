# 跨学科共鸣提取器

> 源自思维模型库中 8 条跨书共鸣和 4 条张力的实战提炼。核心信念：**不同学科谈论同一件事时，往往比单一学科的深入更有洞察力。**

## 分析框架

### 第一步：表面映射（Surface Mapping）

列出两个领域中的概念对应关系。不急于判断深浅，先铺开。

**示例**：
| 领域A：进化生物学 | 领域B：中国哲学 |
|---|---|
| 自然选择 | 天道 |
| 基因的自私 | 人性本恶 |
| 适者生存 | 不争即不败 |
| 突变 | 变通 |

### 第二步：结构同构检测（Structural Isomorphism）

从表面对应中寻找**结构性的同构**——不是"这两个词意思差不多"，而是"这两个系统的运作逻辑在某个抽象层面完全一致"。

检测标准：
1. **因果链同构**：A领域的因果链 X→Y→Z 是否在B领域中有对应的 X'→Y'→Z'？
2. **悖论同构**：A领域中的反直觉发现，是否在B领域中也存在？
3. **失败模式同构**：A领域中的典型错误，是否在B领域也反复出现？

用一句话总结同构：「在 [抽象层面] 上，A的 [机制] 和B的 [机制] 是同一件事。」

### 第三步：张力检测（Tension Detection）

**比共鸣更有价值的是冲突。** 如果两个领域在同一问题上给出了相反的答案，这个张力往往比共识更有学习价值。

检测维度：
1. A说"应该X"，B说"应该非X"——是真正的矛盾，还是层次不同？
2. 如果是层次不同，哪个层次更深？
3. 有没有第三个领域可以调和这个张力？

### 第四步：生成洞见

输出三种类型的洞见：

**🌊 共鸣型**：两个领域从不同角度说同一件事
```
共鸣：[标题]
- 领域A：[表述]
- 领域B：[表述]
- 深层同构：[用一句话说明为什么它们本质上是一回事]
- 实践意义：[知道这个同构，对我们有什么用？]
```

**⚔️ 张力型**：两个领域在同一问题上的有价值的冲突
```
张力：[标题]
- 领域A主张：[...]
- 领域B主张：[...]
- 冲突层面：[它们在哪个层面冲突？]
- 调和路径：[有没有更高层面可以整合？]
- 学习价值：[这个冲突教会我们什么？]
```

**🌟 META型**（当 ≥3 个领域指向同一模式时）：
```
元模式：[标题]
- 领域A：[证据]
- 领域B：[证据]
- 领域C：[证据]
- 为什么这个模式跨越了学科边界：[...]
- 这个模式的边界在哪里（它在什么情况下不成立）：[...]
```

## 质量检查清单

每次提取完成后，自检：

- [ ] 同构是结构性的，而非仅仅是词汇上的巧合？
- [ ] 有没有过度类比？（两个东西表面像但底层逻辑完全不同）
- [ ] 张力是真实的冲突，还是我制造的稻草人？
- [ ] 洞见有实践意义，还是只是智力游戏？
- [ ] 有没有遗漏更深的层次？试着再往下挖一层。

## 与思维模型库的集成

如果用户有思维模型库（cards/ 目录），提取共鸣时应：
1. 检查是否与现有卡片/共鸣重复
2. 如果是新发现，建议创建新卡片或追加到 INDEX.md 的跨书共鸣部分
3. 如果涉及 ≥3 个领域，建议创建 META 元卡片

## 使用示例

**用户输入**：「道德经的'不争'和博弈论的'纳什均衡'有什么关系？」

**预期输出**：先做表面映射（不争↔合作策略），再检测结构同构（两者都指向"在重复博弈中，非攻击性策略的长期优势"），然后检测张力（道德经是无条件不争 vs 博弈论的tit-for-tat是有条件合作），最后给出带调和路径的洞见。
